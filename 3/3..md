# Markdown

### 简介 

---

> `Markdown`是一种轻量级标记语言，创始人为[约翰·格鲁伯](https://www.zhihu.com/topic/19598915/hot)。它允许人们使用易读易写的纯文本格式编辑本党，然后转换成有效的`XHTML`文档。

### 作用

     比如*斜体文本* ，**粗体文本**，***粗斜体文本***。

     比如

     ​	分割线
